# قرار عاشقانه‌ی ما 💕

یک نظرسنجی کوچولوی عاشقانه برای دعوت پارتنر به بیرون — ساخته‌شده با **Vue 3** به صورت **کاملاً استاتیک و آفلاین**.

## ساختار فایل‌ها

```
index.html              ← کل اپ
vendor/vue.global.prod.js   ← باندل محلی Vue 3 (بدون CDN)
fonts/vazirmatn.css         ← @font-face محلی
fonts/Vazirmatn-Variable.woff2  ← فونت محلی (وزن ۱۰۰–۹۰۰)
```

تمام منابع (Vue و فونت) **داخل خود پروژه** قرار دارند و هیچ درخواست اینترنتی‌ای زده نمی‌شود.

## اجرا

به دلیل سیاست CORS مرورگرها برای فونت‌ها، بهتر است از طریق یک سرور محلی باز شود (نه مستقیم `file://`):

```bash
python3 -m http.server 8000
# سپس مرورگر: http://localhost:8000
# و از موبایل: http://<آی‌پی‌سیستم>:8000
```

هیچ build step یا نصبی لازم نیست.

برای تست روی موبایل به صورت لوکال:

```bash
python3 -m http.server 8000
# سپس از موبایل: http://<آی‌پی‌سیستم>:8000
```

## دیپلوی استاتیک

کل پروژه فقط `index.html` است؛ آن را روی هر هاست استاتیکی بگذارید:

- **GitHub Pages**: فایل را در ریشه‌ی ریپو بگذارید و Pages را روشن کنید.
- **Netlify / Vercel / Cloudflare Pages**: پوشه را drag & drop کنید (بدون build command).
- **هر هاست دیگری**: کافیست `index.html` آپلود شود.

## منطق بازی

- **صفحه‌ی مکان**: گزینه‌ها به صورت حباب‌های **ثابت** روی یک شبکه‌ی ۲×۳ نمایش داده می‌شوند (همیشه داخل صفحه). فقط گزینه‌ی **«کله‌پاچه»** قابل انتخاب است؛ به محض **لمس/کلیک** روی هر گزینه‌ی دیگر، آن حباب با یک پرش نرم جایش را با حبابی دیگر عوض می‌کند و انتخاب انجام نمی‌شود. چون رفتار روی خودِ tap فعال می‌شود، روی موبایل (بدون ماوس) هم درست کار می‌کند.
- **صفحه‌ی زمان**: هر چهار گزینه قابل انتخاب‌اند (هیچ تله‌ای برای زمان مشخص نشده بود). 💡 نکته: گزینه‌های «چهارشنبه صبح»، «۲۷ خرداد ۱۴۰۵» و «۱۷ ژوئن ۲۰۲۶» در واقع به یک روز اشاره می‌کنند!
- **صفحه‌ی تشکر**: از شرکت در نظرسنجی تشکر می‌کند و خلاصه‌ی قرار را نشان می‌دهد.

تغییر گزینه‌ی هدف: در `index.html` آرایه‌ی `places` را ببینید و فیلد `target:true` را روی گزینه‌ی دلخواه بگذارید.
